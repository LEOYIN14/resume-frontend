# 个人简历管理系统

一个集成了AI能力的个人知识管理与简历生成系统。

## 功能特性

- 📝 动态简历生成与更新
- 📊 项目管理和文档关联
- 🤖 AI智能分析与内容生成
- 🔒 本地数据存储
- 📱 响应式Web界面

## 技术栈

### 前端
- React 18 + TypeScript
- Redux Toolkit 状态管理
- Ant Design UI组件库
- ECharts 数据可视化
- Vite 构建工具

### 后端
- Node.js + Express
- SQLite 数据库
- JWT 认证
- DeepSeek AI API集成

## 快速开始

### 安装依赖
```bash
npm run install:all
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 项目结构

```
├── frontend/          # 前端React应用
├── backend/           # 后端Node.js服务
├── docs/             # 项目文档
└── package.json      # 根项目配置
```
